verificar versão do node

instalar nestjs cli
    npm install -g @nestjs/cli

criar novo projeto nestjs
    nest new mystika-backend

verificar os modulos necessários

criar modulos/apis completas
    nest g resource <nome-do-modulo>
ou criar module, service e controller manualmente pela cli
    nest g module <nome-module>
    nest g controller <nome-controller>
    nest g service <nome-service>
